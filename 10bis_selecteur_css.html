Propriétés CSS utilisées dans cette page

Décrivez leur usage et les valeurs possibles en fonction de ce que vous déjà vu

  1. background : on peut l'utilisée pour la couleur de fond, il peut prendre plusieurs valeur, ( reapeat, url )
  2. color : changer la couleur de textes peut prendre un nom (en anglais ),RGB ou un hexadécimal ../
  3. float : retir du flux un élément peut prandre les valeurs right et left.
  4. font-family : police du texte, exmemple times new roman.
  5. font-size : permet d'augmeter ou de balisser la taille du texte exemple font-size: 20px (px, %, em, rem)
  6. height : règle la hauteur,( px, %, vh, vw)
  7. left : pour mettre à gauche (px, %,...)
  8. line-height : espacement entre les lignes de texte ( interligne ) (px, %, em, rem)
  9. margin : taille des marge exterieurs, peut prandre jusqu' à 4 valeurs, sens horaire (px, %, em...).
  10. padding : définit la taille des marges intérieur, sens horaire (px,...)
  11. position : permet de changer la position d'une balise, statique, relative, fixed...
  12. text-align : permet d'aligner un texte, center, justify, right, left
  13. text-decoration : permet de retier le soulignement avec la valeur none
  14. top : fonctionne avec position, permet de situer sur l'axe des ordonnées la position'une balise
  15. transition : précise un permamétre 'une animation comme sa durée, peut prandre une valeur en secondes.
  16. width : permet de motifer la larger

Propriétés CSS à rechercher

Recherchez les propriétés ci-dessous, faites des tests avec cette page depuis la console de deduggage (F12 sur Firefox ou clic "inspecter").

  1. font-weight : change l'etat d'une police d'ecriture -> graisse (bold, valeur de 1 a 1000).
  2. letter-specing : pour changer les l'espace entre les caractères
  3. word-spacing : détermine l'espace entre les mots px, em, ...
  4. text-shadow : ombrage du texte, en px, x, y fondu, coloeur.
  5. background-image :modife l'arrire plan d'une balise, prend une url, un dégradé comme valeurs
  6. backgroud-size : définit la taille des images d'arrière-plan pour l'élément (cover, px, %, containe ) .
  7. opacity : défini la transparence de la balise, 0 à 1, 0 -> invisible
  8. boder-style : définir le style de bordure utilisé, dotted, dashed, solide, double, groove, ridage
  9. box-shadow : ajoute une ombre derriére une baliseen px %, x, y, blur, diffustion black 1px 1px 5px 10px
  10. list-style-image : remplace la puce d'une liste par une image, url('')
  11. border : ajoute une bordure, coloeur, taille en px, style de trait
  12. border-radius : arrondie le coin d'un élément, sa valeur est en px, em et ex
  13. border-width : définir la largeur d'une bordure en px, em et ex
  14. font-variant : modife le style de la police, plusieurs valeurs mall-caps, normal...
  15. backgroun-color : caractèrse la couleur du background, hexadécimal, couleur en anglais ...
  16. !important : à mettre à la fin d'une ligne, force le navigateur à utiliser ce style
